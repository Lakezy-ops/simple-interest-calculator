# Contributing to Simple Interest Calculator

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

1. **Fork the Repository**: Create your own copy of this project by clicking the "Fork" button at the top of this page.
2. **Clone the Project**: Download your fork to your local machine.
3. **Create a Branch**: Use a descriptive name for your branch (e.g., `fix-calculation-error`).
4. **Make Your Changes**: Implement your bug fix or enhancement.
5. **Commit and Push**: Save your changes and push them to your GitHub fork.
6. **Submit a Pull Request**: Go to the original repository and click "New Pull Request" to share your changes for review.

## Reporting Bugs
If you find a bug, please open an "Issue" in this repository and provide as much detail as possible about the problem and how to reproduce it.

## Suggesting Enhancements
We are always looking for new ideas! Feel free to open an issue to suggest a new feature or improvement.
